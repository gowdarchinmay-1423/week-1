# Resume Screener - Week 1

## Overview
This is the Week 1 version of the Resume Screener project.  
The focus was on **basic text extraction** from uploaded resumes in PDF, DOCX, and TXT formats using Python.

## Features
- Extract text from PDF, DOCX, and TXT resumes
- Clean and preprocess extracted text
- Streamlit app interface for uploading resumes
- Display extracted text

## Files
- `app.py` – Main Streamlit app
- `README.md` – Project description
- `reqiurements.txt` – Required Python packages
